# X-Blades Trainer
A simple cheat tool for an old game called X-Blades released by Gaijin Entertainment in 2009.

## Keybindings
F1: Infinite Health<br>
F2: Infinite Rage<br>
F3: Add 1,000,000 Souls<br>
F4: Add 10,000,000 Souls<br>
F5: Exit trainer<br><br>
